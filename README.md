# 🤖 Animation Fonctionnement ChatGPT

Une application interactive et pédagogique conçue pour visualiser et comprendre le voyage d'une requête utilisateur vers une Intelligence Artificielle.

![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)

## 🎯 Objectif Pédagogique

L'IA peut souvent sembler magique ou mystérieuse. Ce projet a pour but de **démystifier le fonctionnement de ChatGPT** en décomposant chaque étape de la communication entre l'utilisateur et les serveurs d'IA.

L'animation illustre visuellement :
1. **L'envoi** de la question depuis votre appareil.
2. **Le transit** à travers Internet.
3. **Le traitement** par les serveurs de calcul.
4. **La réflexion** (le "cerveau" de l'IA).
5. **Le retour** de la réponse générée vers l'utilisateur.

## ✨ Fonctionnalités

- 🔄 **Animation pas à pas :** Suivez le trajet de l'information étape par étape.
- 🤖 **Prompto le Robot :** Un petit guide visuel qui transporte vos données.
- 🧠 **Visualisation du Réseau Neuronal :** Une représentation simplifiée de l'IA en train de réfléchir.
- ⏯️ **Mode Lecture Automatique :** Laissez l'animation se dérouler toute seule.
- 📱 **Interface Responsive :** Une expérience fluide sur tous les écrans.

## 🚀 Installation

Pour lancer ce projet sur votre machine locale, suivez ces étapes :

1. **Clonez le dépôt :**
   ```bash
   git clone <votre-url-github>
   cd <nom-du-dossier>
   ```

2. **Installez les dépendances :**
   ```bash
   npm install
   ```

3. **Lancez le serveur de développement :**
   ```bash
   npm run dev
   ```

4. **Ouvrez votre navigateur :**
   Rendez-vous sur `http://localhost:3000`

## 🛠️ Technologies

Ce projet utilise des technologies modernes du web :
- **React 19** - Pour l'interface utilisateur.
- **Tailwind CSS** - Pour un design élégant et rapide.
- **Framer Motion** - Pour des animations fluides et interactives.
- **Lucide React** - Pour les icônes.
- **Vite** - Pour un environnement de développement ultra-rapide.
